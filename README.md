## Chess openings
A small application to store openings in a home-brewed recurisve file system. These openings can then be replayed in the "Explorer" portion of the application. 


## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

```bash
yarn build
yarn start
```
